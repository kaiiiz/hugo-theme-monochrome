Icon

Monochrome built-in icons.

Usage 

{{< icon vendor="" name="" link="" >}}

Parameters:

  • vendor: Icon vendor (Supported value: feather, simple)
  • link: Render link for icon
  • name: Icon name

Available Icons

namesvg

moon
