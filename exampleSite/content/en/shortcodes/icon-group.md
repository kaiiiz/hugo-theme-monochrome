# Icon Group

Wrap icons with grid layout

## Usage

```
{{</* icon-group */>}}

{{</* /icon-group */>}}
```

## Examples

```html
{{</* icon-group */>}}
{{</* icon name="github" */>}}
{{</* icon name="rss" */>}}
{{</* icon name="twitter" */>}}
{{</* /icon-group */>}}
```

{{< icon-group >}}
{{< icon name="github" >}}
{{< icon name="rss" >}}
{{< icon name="twitter" >}}
{{< /icon-group >}}
